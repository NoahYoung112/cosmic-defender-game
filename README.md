# Cosmic Defender

## Project Description
Cosmic Defender is a Java arcade-style shooter built using object-oriented programming principles in BlueJ. Players defend the galaxy by piloting a spacecraft and destroying incoming enemies while collecting power-ups to survive longer.

The game features:
- Three enemy types: Fast, Normal, and Tank
- Two power-up types: Health and Speed
- Inheritance-based object design for enemies, player, projectiles, and power-ups
- Real-time action with score tracking and increasing challenge

## Technologies Used
- Java
- BlueJ
- Object-oriented programming
- Inheritance, polymorphism, and encapsulation

## How to Run
1. Open BlueJ.
2. Load the `cosmic-defender-game` project folder.
3. Compile all classes.
4. Run the `GameFrame` or `GamePanel` class to start the game.

> Note: Ensure all Java source files are present in the project and compiled successfully before launching the game.
